| 浏览

Golang（简称Go）是由 Google的 Robert Griesemer，Rob Pike 及 Ken Thompson开发的一种静态强类型、编译型、并发型，并具有垃圾回收功能的编程语言。

第一个程序

创建golang_study目录用来存放我们的项目文件

lighthouse@VM-8-13-ubuntu:~$ mkdir golang_study
lighthouse@VM-8-13-ubuntu:~$ ls
golang_study

切换到golang_study目录下，创建hello目录用来存放第一个程序的代码文件

lighthouse@VM-8-13-ubuntu:~$ cd golang_study/
lighthouse@VM-8-13-ubuntu:~/golang_study$ mkdir hello
lighthouse@VM-8-13-ubuntu:~/golang_study$ ls
hello

切换到hello目录下，使用touch命令创建hello.go文件

lighthouse@VM-8-13-ubuntu:~/golang_study$ cd hello/
lighthouse@VM-8-13-ubuntu:~/golang_study/hello$ touch hello.go
lighthouse@VM-8-13-ubuntu:~/golang_study/hello$ ls
hello.go

使用Vim编辑hello.go文件

package main

import(
	"fmt"
)

func main(){
    fmt.Println("Hello,World!")
}

在Vim中按i键进入编辑，编辑好后按Esc键退出编辑，输入:wq保存并退出Vim

在命令行输入go mod init hello创建模块hello

lighthouse@VM-8-13-ubuntu:~/golang_study/hello$ go mod init hello
go: creating new go.mod: module hello
go: to add module requirements and sums:
        go mod tidy

运行go语言程序有两种方式：

使用go run命令直接运行

lighthouse@VM-8-13-ubuntu:~/golang_study/hello$ go run hello.go
Hello,World!

使用go build编译后再运行可执行文件

lighthouse@VM-8-13-ubuntu:~/golang_study/hello$ go build hello.go
lighthouse@VM-8-13-ubuntu:~/golang_study/hello$ ls
go.mod  hello  hello.go
lighthouse@VM-8-13-ubuntu:~/golang_study/hello$ ./hello 
Hello,World!

Go程序的基本结构

上一节我们编写了第一个程序hello.go

package main

import(
	"fmt"
)

func main(){
    fmt.Println("Hello,World!")
}

现在我们就通过这个程序来了解一个Go程序的基本结构:

  1. 包名定义
package main

package关键字用来定义一个包名，一般是当前代码文件的父目录的名称，main是一个特殊的名称，每个可独立执行的程序都要包含一个名为main的包。

  1. 模块导入
//推荐写法，方便后续添加
import(
"fmt"
)

//只需要导入一个包时，可简写为
import "fmt"

import告诉Go编译器当前程序需要导入那些模块（包），fmt包是Golang自带的标准库，实现了格式化IO的函数。

  1. 程序入口
func main(){
//statement
}

func关键字是Golang用来声明函数的，而main()是一个特殊的函数，是Golang程序开始执行的函数，是一个程序的入口。

  1. 语句编写
fmt.Println("Hello,World!")

在main()函数体内可以编写我们所要实现的功能代码，上面的代码就是使用fmt包中的Println()函数输出Hello,World字符串显示在屏幕上。
