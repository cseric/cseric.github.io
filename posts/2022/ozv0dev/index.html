C++学习笔记 | 常量

2022-07-10
| C++
| 浏览
| 1363 字

有时我们希望定义一些固定不变的数据，这时我们就可以使用C++给我们提供的const关键字对变量进行修饰，使其值不能再改变。

常量介绍

常量在定义时必须初始化，格式如下：

const dataType constantName = value;

const int value = 64;	// OK 编译期初始化
const int value2 = get_arr_size();	// OK 运行期初始化
const int value3;	// ERROR: 常量定义时必须初始化

int num = 32;
const int value4 = num;	// OK 将num的值拷贝给value4
int result = value;	// OK 将value4的值拷贝给result

const是常量对自身的行为约束，只有在涉及修改自身的值时，const才会发挥作用，用对象初始化或者常量作为对象的初值都是进行值拷贝操作，不会改变自身。

常量作用范围

默认情况下，常量只在当前文件内有效；若要在文件外使用，可以在声明和定义时都使用extern关键字修饰：

// 定义并初始化
extern const int value = 64;

// 声明
extern const int value;

顶层const和底层const

顶层const表示对象是常量，对于任何数据类型都适用；底层const则是对于指针和引用等复合类型的基本类型有关；指针类型既可以是顶层const，也可以是底层const：

int value = 64;
int* const ptr = &value;	// 不能改变ptr的指向，属于顶层const
const int value2 = 32;		// 常量，属于顶层const
const int* cptr = &value2;	// cptr的指向可以改变，但值不可变， 属于底层const
const int* const ccptr = cptr;	// 前一个const是底层const，后一个const是顶层const
const int &ref = value2;	// 常量引用都是底层const

执行对象的拷贝操作时，顶层const不受影响，但两个对象的底层const必须相同或者两个对象的数据类型可以相互转换，非常量可以转换为常量：

value = value2;	// OK 将value2的值拷贝给value，顶层const不受影响
cptr = ccptr;	// OK cptr和ccptr指向的类型相同，ccptr的指向(顶层const)不受影响
int *ptr2 = ccptr; // ERROR: ptr2没有底层const
cptr = &value;	// OK 常量指针可以指向非常量
int &ref2 = value2;	// ERROR: 普通引用不能绑定常量
const int &ref3 = value;	// OK 常量引用可以绑定非常量

constexpr

constexpr是C++11标准引入的一个关键字，允许将变量声明为constexpr类型并由编译器来验证变量的值是否是一个常量表达式。声明为constexpr的变量一定是一个常量，而且必须用常量表达式初始化：

constexpr int get_size(){
    return 10;
}

int main() {
    constexpr int value = 20;			// OK
    constexpr int value2 = value + 20;	 // OK
    constexpr int size = get_size();	 // 此处的get_size()必须是constexpr函数
    return 0;
}

常量表达式是指值不会改变并在编译期就能得到计算结果的表达式，字面值和用常量表达式初始化的const对象也是常量表达式；字面值包括算术类型、引用和指针等，自定义class、string类型等不属于字面值类型。

指针和引用可以定义成constexpr，但是初始值必须是固定的，constexpr指针的初始值可以是nullptr或者0以及某个固定地址的对象，constexpr引用的初始值一般是某个固定地址的对象。函数体之内的对象不是固定地址，而函数体之外的对象是存放在固定地址上，所以函数体之外的对象可以用来初始化constexpr指针；但函数体内也有一种拥有固定地址的对象，即局部静态对象，所以constexpr引用和constexpr指针可以使用局部静态对象来初始化。

constexpr关键字修饰指针时，仅对指针本身有效，即为指针的顶层const；constexpr指针和其他常量指针类似，既可以指向常量也可以指向非常量。
