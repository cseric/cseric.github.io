Ubuntu20.04更换国内源教程

2022-08-10
| Linux
| 浏览
| 584 字
| 2 分钟

安装Ubuntu系统之后，第一件事就是更换国内源加快程序和依赖的下载和更新加速。

备份sources.list文件

要修改一个文件，首先要对其先备份，如果出错可以及时恢复默认状态：

sudo cp -p /etc/apt/sources.list /etc/apt/sources.list.bak

编辑sources.list文件

sudo vim /etc/apt/sources.list
// or
sudo gedit /etc/apt/sources.list

更新国内源

把上面的任意一个源复制到/etc/apt/sources.list文件中保存，使用以下命令更新：

sudo apt update

命令执行完成后，就可以根据自己的需要安装软件了，例如：

sudo apt install vim
