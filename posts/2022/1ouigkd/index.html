Golang基础学习十四 | 接口

2022-08-06

Go语言提供了接口数据类型，接口就是将所有具有共性的方法定义在一起，其他类型只要实现了这些方法就实现了这个接口。

接口的定义

使用type和interface关键字定义接口

type Car interface {
    driver()
}

接口的使用

package main

import "fmt"

type Car interface{
	driver()
}

type BMW struct {
}

type Audi struct {
}

func (bmw *BMW) driver() {
	fmt.Println("BMW正在以40km/h速度行驶!")
}

func (audi *Audi) driver() {
	fmt.Println("Audi正在以45km/h速度行驶!")
}

func main(){
	var car Car
	car = new(BMW)
	car.driver()

	car = new(Audi)
	car.driver()
}

执行结果为：

BMW正在以40km/h速度行驶!
Audi正在以45km/h速度行驶!

方法

上节示例中使用了Go语言中的一种新语法，被称作方法，即带接收者的函数，格式如下：

// 接收者的类型可以是值类型或指针类型
func (receiverName receiverType) methodName([parameter list]) [returnValue list] {
}

只要有一个指针实现，定义变量时就必须是指针变量

package main

import "fmt"

type Car interface{
	driver()
}

type BMW struct {
}

type Audi struct {
}

func (bmw *BMW) driver() {
	fmt.Println("BMW正在以40km/h速度行驶!")
}

func (audi *Audi) driver() {
	fmt.Println("Audi正在以45km/h速度行驶!")
}

func driver(car Car) {
	car.driver()
}

func main(){
	var car Car
	car = new(BMW)
	car.driver()
	driver(car)

	// car = Audi{}	// ERROR: 结构体Audi实现Car接口的方法中有指针实现
	car = &Audi{}
	car.driver()
	driver(car)
}
