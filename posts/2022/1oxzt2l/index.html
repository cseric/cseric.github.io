Golang基础学习二 | 语法风格和规范

2022-07-29
| Golang
| 浏览
| 393 字
| 1 分钟

Go语言非常简洁，但是它也有些严格，它有一套自己的语法风格和规范。

代码规范

  1. 语句结束时不需要加分号
fmt.Println("Hello,Go!")
fmt.Println("Hello,Golang!")
  1. 大（花）括号不能换（单）行放置
//正确写法
func main(){
//statement
}

//错误写法
func main()
{
//statement
}
  1. if语句和for语句的条件不需要小括号包裹
//if语句
if age > 18 {
//statement
}

//for语句
for i := 0; i < 10; i++ {
//statement
}

Go内置gofmt工具，能够自动整理代码中多余空白、对齐变量名以及对齐空格（转换成Tab）

注释

Go语言注释分为两种：

单行注释

//这是单行注释

多行注释

/*
这是多行注释
测试文字
*/

标识符

标识符用来命名变量、类型等。标识符由一个或多个字母、数字以及下划线组成的序列，但不能以数字开头。

//以下为有效的标识符
username	userName	user_name
_username	user_123	user123
u12se45r

//以下为无效的标识符
1ab		
1_ab	
a+b

Go语言的关键字也不能作为标识符
