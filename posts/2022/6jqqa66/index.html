Go语言也有指针，但相对于C/C++的指针来说，Go的指针更为简单，特别要注意的一点就是Go语言没有指针运算。

指针介绍

指针就是一个变量指向一段合法的内存地址，对其解引用（*）就可访问该地址中存储的数据，而要获得一个变量的地址就要使用取地址符（&）来获取：

var num int = 6
// 获取变量num的地址赋值给ptr指针
var ptr *int = &num
// 对ptr进行解引用，*ptr相当于获取num变量的值
fmt.Println(*ptr)

指针声明

一个指针变量指向了一个值的内存地址，和变量常量一样，在使用指针前要先声明指针，声明格式如下：

var ptrName *ptrType

*号代表要声明的是指针变量

指针使用

package main

import "fmt"

func main(){
	var num int = 6
	var ptr *int = &num
	
    // 比较两者的地址
	fmt.Println(&num)
	fmt.Println(ptr)
	
    // 比较两者的值
	fmt.Println(num)
	fmt.Println(*ptr)
}

执行结果为：

0xc00000a0c8
0xc00000a0c8
6
6

指针本身也是变量，也有内存地址：

// 打印指针变量的地址
fmt.Println(&ptr)

空指针

当一个指针只有声明没有定义时，即没有指向变量时，它的值为nil，nil和NULL一样指代零值或者空值：

package main

import "fmt"

func main(){
	var ptr *int

	fmt.Printf("%x\n", ptr)
	//fmt.Println(*ptr)	// ERROR: 指针未指向变量，对其解引用会panic
	fmt.Println(ptr == nil)
	fmt.Println(ptr != nil)
}

执行结果为：

0
true
false

指针数组

数组也可以用来存储指针类型，指针数组中的每个元素都保存（指向）一个变量的内存地址，指针数组的声明格式如下：

var ptr [SIZE]*int

示例：

package main

import "fmt"

func main(){
	var num = [6]int{4,8,34,67,23,12}
	var ptr [6]*int

	for i := 0; i < 6; i++ {
		ptr[i] = &num[i]
	}

	for i := 0; i < 6; i++ {
		fmt.Println(*ptr[i])
	}
}

执行结果为：

4
8
34
67
23
12

数组指针

指针也可以指向一个数组，数组指针和指针数组有时很容易就会搞混，数组指针的声明格式如下：

var ptr *[SIZE]int

示例：

package main

import "fmt"

func main(){
	var num = [6]int{4,8,34,67,23,12}
	var ptr = &num

	for i := 0; i < 6; i++ {
		fmt.Println(ptr[i], " : ",(*ptr)[i])
	}
}

执行结果如下：

4  :  4
8  :  8
34  :  34
67  :  67
23  :  23
12  :  12

指向指针的指针

一个指针可以指向另一个指针，另一个指针则指向一个变量，就称这个指针是指向指针的指针，指向指针的指针变量声明格式如下：

var pptr **int

示例：

package main

import "fmt"

func main(){
	var num int = 6
	var ptr *int = &num
	var pptr **int = &ptr

	fmt.Println(num)
	fmt.Println(*ptr)
	fmt.Println(**pptr)
}

执行结果为：

6
6
6

指针作为函数参数

指针也可以作为函数的参数。使用指针作为函数参数时，在函数内执行的修改操作会影响实际参数：

package main

import "fmt"

func swap(a *int, b *int)  {
	var temp int
	temp = *a
	*a = *b
	*b = temp
}

func main(){
	a, b := 3, 4

	fmt.Printf("Before: a=%d, b=%d\n", a, b)
	swap(&a, &b)
	fmt.Printf("After: a=%d, b=%d\n", a, b)
}

执行结果为：

Before: a=3, b=4
After: a=4, b=3
